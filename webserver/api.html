API REST COMMANDS

Node & Wallet Name: {{.NodeName}}

HOME

BLOCKCHAIN

/showblockchain Shows the blockchain

/showblock/0 Show a particular blockID in the blockchain

LOCKED BLOCK

/showlockedblock Show the lockedBlock

/appendlockedblock Appends the lockedBlock to the blockchain (Step 2) - Must be Founder

PENDING BLOCK

/showpendingblock Show the pendingBlock

/resetpendingblock Resets the pendingBlock (Step 3) - Must be Founder

/lockpendingblock Moves the pendingBlock to the lockedBlock (Step 1) - Must be Founder

NODELIST

/shownodelist Show the nodelist

/shownode/0 Show a particular index in nodeList

/showthisnode Show thisNode

WALLET ({{.NodeName}})

/showwallet Show this wallet (Public & Private Keys)

/showjeffcoinaddress Show this Address

/showbalance Show the balance for this wallet

/transactionrequest/aaa,bbb/23,44 Request to transfer 23 coins to address aaa & 44 coins to address bbb
This API link not enabled here because it will transfer coins and this action is irreversible. Use the Send Page.

WALLET (Other)

/showaddressbalance/1234 Show balance for the Address 1234
